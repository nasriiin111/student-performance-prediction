# Student Performance Prediction

## Objective
Analyze factors affecting student performance and predict overall scores.

## Dataset
Student Performance Dataset

## Features Used
- Study Hours
- Attendance Percentage
- Age
- Math Score
- Science Score
- English Score

## Exploratory Data Analysis
Key findings:
- Study hours strongly correlate with overall score.
- Attendance has a moderate effect.
- Age has little impact.

- Scikit-learn
